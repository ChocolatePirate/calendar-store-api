#Weather API

In the JSON returned by the API yo'll notice a url that looks like
```
http://{your_name}.schedjoules.com/cities/full/%TEMP%/%LANGUAGE%/%RAIN%/%WIND%/%TIME%/%CITY%.ics&x={your_app_code}
```

The parameters 

Temp

Language

Rain

Wind

Time

City

Parameters can be found at:

```
http://{your_name}.schedjoules.com/cities/full/%TEMP%/%LANGUAGE%/%RAIN%/%WIND%/%TIME%/%CITY%.ics&x={your_app_code}
```
